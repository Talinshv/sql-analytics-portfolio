# sql-analytics-portfolio
# **Database Building**
## 1. **In Github**
### **Create New Repository**
   - Choose New Repository
   - Choose a name for Repository (ex. sql-analytics-portfolio)
   - Visibility should be public
   - Add README file
   - Choose NON for gitignor
   - Create Repository
   - Copy URL
     
## 2. **On your PC** 
### **Clone the Repository**
   - Create a folder and giva name (ex. ACA)
   - Open terminal there and you will see ~/aca
   - Type in terminal " git clone (paste here URL)" ➡️in your ACA folder will be created "sql-analytics-portfolio" folder
   - Type " cd sql-analytics-portfolio " ➡️will go to ACA\sql-analytics-portfolio
   - 
## 3. **In VS code** 
### **Add ACA folder in VS code**
   - Open new folder
   - Go to the ACA\sql-analytics-portfolio folders and select the folder➡️we will have sql-analytics-portfolio folder in VS code
   - Open VS code terminal➡️you are in ACA\sql-analytics-portfolio
